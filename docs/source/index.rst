Welcome to Evidence Extractor's documentation!
==============================================

``evidence-extractor`` is a research software project for semi-automated evidence extraction from scholarly articles using Large Language Models.

This documentation provides details on the internal API of the library. For a user-facing guide, please see the main `README.md` file in the repository.
